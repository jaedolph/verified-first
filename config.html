You must be an affiliate or partner to use this extension as it relies on channel points


1. Configure "First" channel points reward

Create a new channel points reward called "First" (or something similar).

Viewers can claim this reward to show that they were the first to get to your stream.

Set the cost to 1 point. Tick "Cooldown & Limits" and set "Limit Redemptions Per Stream" to 1 so that only one viewer can claim it per stream.

If you already have a reward set up like this, you don't need to create a new one.

Click here to configure a new reward.
